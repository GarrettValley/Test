Focus

For AILA, we concentrated heavily on providing a product that would have a diverse and wide-spread impact on immigrants across the world. We did this through collecting extensive data as well as having as many choices for professions and ethnicity as we could find data for.

Technologies

Data provided is from the 2010 US Census, The Numbeo Database, and other publicly available resources. The back-end was written using JavaScript, and the front-end was created using WebFlow.
